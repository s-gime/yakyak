## 🚀 Development

```bash
# Setup
git clone https://github.com/gaba-dev/yakyak.git
cd yakyak

# Install dependencies
npm install
pip install -r requirements.txt

# Configure
cp .env.example .env

# Run
npm run dev
python backend/server.py
```
